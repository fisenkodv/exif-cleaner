
# Exif Cleaner

## Create Environment

- Install *virtualenv*

```bash
[sudo] pip install virtualenv
 ```

- Create an environment

```bash
virtualenv venv
```

- Activate script

```bash
source venv/bin/activate
```

- Install dependencies

```bash
pip install -r requirements.txt
```

- Deactivate script

```bash
deactivate
```
